# dbt Modern Warehouse

A modern ELT data warehouse built with dbt Core and PostgreSQL.

## Overview

This project implements a modern data warehouse using the Jaffle Shop dataset — a fictional coffee shop. It demonstrates ELT pipeline design, dimensional modeling, data quality testing, and SCD2 snapshots using dbt Core.

## Tech Stack

- **dbt Core** 1.8.2
- **PostgreSQL** 18
- **Python** 3.11
